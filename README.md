**Academy-coder-corner**

Este es un maquetado pensado para aquellas personas que deseen realizar una academia gratuita donde
puedan usar sus experiencias y conocimientos para compartirlas con quienes quieran aprender.


## ¿Qué tecnologías se utilizan en este proyecto?

Este proyecto está desarrollado con:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
